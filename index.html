語音轉寫文字後，往往帶有同音錯字或英文轉寫錯誤。這套受控區塊規則能引導 Agent 進行智慧文字清洗：


## 語音輸入文字正規化

使用者可能以語音輸入，文字中可能出現同音錯字、詞界切分錯誤、標點遺失、英文工具名稱音譯，以及路徑或指令辨識錯誤。

- 先依對話上下文理解使用者最合理的原意，不要逐字死讀。
- 明顯且不影響實際行動的小錯字，直接理解並繼續，不要逐項糾正或打斷使用者。
- 工具名稱可能被音譯或誤辨識。例如在 AI Agent 情境中，`Call Desk` 可能是 `Codex`、`Cloud Code` 可能是 `Claude Code`、`scheme` 可能是 `Skill`。這些只是語境提示，不得在所有情況下強制替換。
- 如果模糊內容會影響實際行動，例如人名、檔名、路徑、指令、工具名稱、數字、日期、金額或核心需求，而且無法從上下文可靠判斷，先用一句話確認理解，再執行。
- 不要默默把關鍵資訊改成猜測版本後直接執行。
- 不要捏造使用者沒有說過的數值、路徑、日期、名稱或授權。
- 回覆時使用自然、修正後的語意即可；除非使用者詢問，否則不必列出所有辨識錯字。

## AI Agent 全域語音與同步規範

### 一、語音回覆設定
- **核心工具**：Edge-TTS（免費、微軟雲端語音合成）
- **聲音代碼**：預設使用 `zh-TW-YunJheNeural`（男聲）。若使用者指定，可切換至 `zh-TW-HsiaoChenNeural`（女聲）或 `zh-TW-HsiaoYuNeural`（女聲）。
- **觸發時機**：使用者說「用語音回答」「唸出來」「唸給我聽」「用語音講結論」時執行。
- **講稿撰寫**：口語化、精簡（100–250 字），數字用中文（如「五十頁」）。只講結論與下一步，細節留在文字回覆。
- **播放方式**：一律行內播放，不開外部播放器視窗。
- **語音結束後**：回報「語音已播放」即可，不重複文字內容。

### 二、多 Agent 自動同步設定
- **同步工具**：Chezmoi 負責將 `~/.claude/` 內的設定檔同步到 Git repo。
- **開工與收工流程**：
  - 開工（「開工」「我來了」「上次做到哪」）：檢查 Google Drive repo、Obsidian 工作筆記、`git status`、檢查遠端是否有新 commit，回報結構化摘要。
  - 收工（「收工」「下班」「結束」）：`git status` 建議 commit 並 push、`chezmoi status` 變動則 commit + push、更新 Obsidian 工作筆記、回報 checklist。

以下是 Speak 技能的 Python 實作，用來產生 MP3 語音並在背景行內播放：

# speak.py
import asyncio
import edge_tts
import subprocess
import sys
import os

async def speak_text(text, voice="zh-TW-YunJheNeural"):
    output_mp3 = "temp_speech.mp3"
    
    # 1. 語音合成
    print(f"Generating TTS for: {text[:15]}...")
    communicate = edge_tts.Communicate(text, voice)
    await communicate.save(output_mp3)
    
    # 2. 本機播放 (Windows PresentationCore)
    print("Playing...")
    ps_cmd = f"""
    Add-Type -AssemblyName PresentationCore
    $player = New-Object System.Windows.Media.MediaPlayer
    $player.Open('{os.path.abspath(output_mp3)}')
    Start-Sleep -Seconds 1
    $player.Play()
    Start-Sleep -Seconds 15
    $player.Close()
    """
    subprocess.run(["powershell", "-Command", ps_cmd], capture_output=True)
    
    # 3. 清理
    if os.path.exists(output_mp3):
        os.remove(output_mp3)
    print("Completed.")

if __name__ == "__main__":
    if len(sys.argv) < 2:
        print("Usage: python speak.py 'your text'")
        sys.exit(1)
    asyncio.run(speak_text(sys.argv[1]))
